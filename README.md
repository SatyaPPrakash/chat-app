## Installation

### Prerequisites
- Node.js (v14 or higher)
- npm (or yarn)

### Steps

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd chat-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the application**
   ```bash
   npm run dev
   ```

4. **Access the app**
   Open [http://localhost:5173](http://localhost:5173) in your browser.
